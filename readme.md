#  Редактор изображений — Вариант 3

**Учебная практика (ознакомительная). Вариант 3**

Разработано: Гетманов И.В., группа КИ25-17/2Б, кафедра Програмная инженерия

Приложение для загрузки, обработки и отображения изображений с использованием Python, OpenCV и Tkinter.

---

## Описание проекта

Это десктоп-приложение позволяет пользователю:

- Загружать изображения форматов PNG и JPG
- Делать снимки с веб-камеры
- Отображать отдельные цветовые каналы (красный, зелёный, синий)
- Добавлять рамку к изображению, толщина рамки задаётся пользователем.
- Переобразовывать изображение в инверсивное ему
- Рисовать зелёную линию по координатам
- Сохранять изображение на диске

Приложение удобно в использовании и снабжено обработкой ошибок с понятными сообщениями.

---

## Технологии

- Python 3.10
- OpenCV
- Pillow
- NumPy
- Tkinter (входит в стандартную библиотеку Python)

## Установка и запуск

1. Склонируйте репозиторий или скачайте исходный код проекта.

2. Перейдите в папку с проектом 

3. Создайте и активируйте виртуальное окружение:

- **Windows:**
  ```bash
  python -m venv venv
  source venv\Scripts\activate
  ```

- **Linux / macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Запустите приложение:
```bash
python project_app.py
```

## Использование

- **Загрузить изображение** — выбрать файл PNG или JPG.
- **Сделать снимок с веб-камеры** — сделать фото с подключённой камеры.
- **Показать канал** — выбрать и отобразить красный, зелёный или синий канал.
- **Добавить границы** — добавляет рамку к изображению, ширина рамки задаётся пользователем.
- **Эффект: Негатив** — преобразовывает изображение в инверсивное ему.
- **Нарисовать линию** — рисует зелёную линию по координатам.
- **Сохранить как..** - Сохряняет файл в выбранной папке
- **Панель состояния (Status Bar):** Вывод логов и предупреждений о совершенных операциях в реальном времени.
