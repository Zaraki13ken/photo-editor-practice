#  Редактор изображений — Вариант 3

**Учебная практика (ознакомительная). Вариант 3**

Разработано: Гетманов И.В., группа КИ25-17/2Б, кафедра Програмная инженерия

Приложение для загрузки, обработки и отображения изображений с использованием Python, OpenCV и Tkinter.

---

## Описание проекта

Это десктоп-приложение позволяет пользователю:

- Загружать изображения форматов PNG и JPG
- Делать снимки с веб-камеры
- Отображать отдельные цветовые каналы (красный, зелёный, синий)
- Добавлять рамку к изображению, толщина рамки задаётся пользователем.
- Переобразовывать изображение в инверсивное ему
- Рисовать зелёную линию по координатам
- Сохранять изображение на диске

Приложение удобно в использовании и снабжено обработкой ошибок с понятными сообщениями.

## Установка и запуск

1. Склонируйте репозиторий или скачайте исходный код проекта.

2. Перейдите в папку с проектом
   
   2.1. Для обладателей Linux системы установите пакет Tkinter:
   
   - **Ubuntu / Debian / Linux Mint:**
     ```Bash
     sudo apt update
     sudo apt install python3-tk
     ```
     
   - **Fedora / RedHat:**
     ```Bash
     sudo dnf install python3-tkinter
     ```
     
   - **Arch Linux / Manjaro:**
     ```Bash
     sudo pacman -S tk
     ```

3. Создайте и активируйте виртуальное окружение:

- **Windows:**
  ```bash
  python -m venv venv
  source venv\Scripts\activate
  ```

- **Linux / macOS:**
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  ```

4. Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

5. Запустите приложение:
    ```bash
    python project_app.py
    ```

## Использование

- **Открыть файл PNG/JPG** — выбрать файл PNG или JPG для загрузки в приложение.
- **Сделать снимок с веб-камеры** — сделать фото с подключённой камеры и использовать его в качестве изображения для обработки.
- **Показать канал** — выбрать и отобразить красный, зелёный или синий канал изображения.
- **Добавить границы** — добавляет рамку к изображению, ширина рамки задаётся пользователем в поле "размер границ (пикс)".
- **Эффект: Негатив** — преобразовывает изображение в инверсивное ему.
- **Нарисовать линию** — рисует зелёную линию на изображении по координатам с указанием толщины линии и координат нач. и кон. точки(X1,Y1 — координаты начальной точки; X2,Y2 — координаты конечной точки)
- **Сохранить как..** — Сохряняет файл в выбранной папке
- **Панель состояния (Status Bar):** Вывод логов и предупреждений о совершенных операциях в реальном времени.

## Пример использования приложения

### 1. Загрузка изображения

- Нажмите кнопку "Открыть файл PNG/JPG" и выберите любое изображение с расширением .png или .jpg

### 2. Выбор цветового канала

- Нажмите кнопку любого цветового канала, например **"Синий канал (B)"**, если хотите можете вернуть оригинальный цвет через кнопку **"Оригинальный цвет"**

### 3. Инверсия

- Для инверсии нажмите на кнопку **"Эффект: Негатив"**

### 4. Добавление границ

- Укажите в окне с названием **"Размер границ (пикс)"** толщину границ в пикселях, например `20`
- Нажмите на кнопку **"Добавить границы"**

### 5. Отрисовка зелёной линии
- В ведите в поле **X1** и **Y1** координаты начальной точки, например **X1:** `10` **Y1:** `10`
- В ведите в поле **X2** и **Y2** координаты конечной точки, например **X2:** `200` **Y1:** `200`
- Нажмите на кнопку **"Нарисовать зелёную линию"**, чтобы нарисовать зелёную линию по заданным вами ранее координатам

### Итог 
- Вы получили изображение с синим цветовым каналом, цвета которого инверсировали, были добавлены границы толщиной в `20` пикселей, была нарисована линия по координатам **X1:** `10` **Y1:** `10` ; **X2:** `200` **Y1:** `200`
